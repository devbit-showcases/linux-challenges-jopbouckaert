# Public key script

> A script to add your public key to the authorized keys

Insert your github username and your key is added to the authorized keys of your device.

## Run

Run the script with following command:

```bash
bash ./script.sh
```

OR make it executable:

```bash
chmod +x ./script.sh
./script.sh
```
